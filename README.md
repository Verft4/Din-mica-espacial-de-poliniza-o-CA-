# Din-mica-espacial-de-poliniza-o-CA-
# Dinâmica espacial de polinização e fluxo de pólen: uma abordagem baseada em Autômatos Celulares

Simulação  da dinamica espacial  de polinização das abelhas  - Automatos Celulares(CA) 


Esse Repositorio tem a implementação de um modelo de Automatos Celulares para simular ,compreender e prever a área de influência de uma colmeia e a dinâmica de transferência de pólen.
